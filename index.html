class ConnectionObservable(
    private val context: Context
) {

    private val observers = mutableListOf()

    fun addObserver(observer: ConnectionObserver) {
        observers.add(observer)
    }

    fun onConnected() {
        observers.forEach {
            it.onConnected()
        }
        Toast.makeText(context, "CONNECTED", Toast.LENGTH_SHORT).show()
    }

    fun onDisconnected() {
        observers.forEach {
            it.onDisconnected()
        }
        Toast.makeText(context, "DISCONNECTED", Toast.LENGTH_SHORT).show()
    }
}

Проекты

  • RanobeReader

    Небольшое клиент-серверное приложение для чтение лайт-новелл. Вся информация берется с сайта ранобэ.рф. Там есть api с помощью которого всю информацию можно получить в JSON формате.

    Для взаимодествия с api используется библиотека Retrofit, а для парсинга JSON - Jackson.

    Для сохранения книг в избранное использую базу данных, но работаю не напряму с SQL, а с помощью библиотеки Room.

    За основу взят архитектурный паттер MVVM. Так же используются и другие компоненты Android Jetpack. Например Navigation и Paging library.

    Для работы с данными из Room и Retrofit использовался Coroutines Flow Adapter.

    Использую Dagger2 для Dependecy Injection.

  • IntelliDeck

    Приложение для взаимодействия с Intellij Idea. Позволяет добавлять различные конфигурации запуска/дебага приложения. Для передачи данных между приложением и плагином используется TCP сервер и клиент.
